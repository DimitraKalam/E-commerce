# E-commerce
Undergraduate project for the elective course e-business

Κατάστημα πώλησης αθλητικού εξοπλισμού για Σκι.

      Υλοποίηση ενός ελληνικό ηλεκτρονικού καταστήματος χρησιμοποιώντας την open- source πλατφόρμα Wordpress (www.wordpress.org) και την επέκταση αυτής 
      Woocommerce (www.woocommerce.com). Το ηλεκτρονικό κατάστημα παρέχει το σύνολο της βασικής λειτουργικότητας που υποστηρίζει η πλατφόρμα για την 
      παρουσίαση του καταλόγου προϊόντων και τη διαδικασία υποβολής παραγγελιών.
